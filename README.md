# ACM 学习仓库

这是一个用于整理 `ACM / 算法竞赛 / C++` 学习笔记的知识仓库，内容以 `Markdown` 为主，适合在 `Obsidian/typora` 等 `markdown` 编辑器中阅读和维护。

仓库当前主要覆盖三部分内容：

- `CPP`：C++ 基础、面向对象、现代 C++、编译与构建等内容
- `STL`：常用标准库容器与基础用法
- `传统算法`：算法竞赛中常见的基础算法、数据结构、图论、动态规划、数论、字符串等专题

## 内容说明

### 1. CPP

主要记录 `C++` 学习过程中的基础知识与语言特性，包括：

- 类与对象
- 构造函数、拷贝、初始化列表
- 友元、继承、多态
- 运算符重载
- 文件操作
- `Lambda` 等现代 C++ 特性
- 编译流程与 `Makefile` 基础

### 2. STL

记录竞赛中高频使用的标准模板库内容，例如：

- `vector`
- `queue`
- `priority_queue`

后续会继续补充 `set`、`map`、`deque`、`stack`、`algorithm` 等常用组件。

### 3. 传统算法

这是仓库的核心部分，主要整理常见算法专题，包括但不限于：

- 基础算法：排序、贪心、高精度
- 动态规划：线性 DP、树形 DP、概率 DP、背包问题
- 图论：最短路、最小生成树、拓扑排序、LCA、连通性、树上问题
- 数据结构：并查集、堆、树、树状数组、线段树、平衡树、单调栈、单调队列
- 字符串：KMP、Trie、字符串哈希、AC 自动机
- 数论：GCD/LCM、快速幂、模运算、质数判定、组合数、位运算
