# 最长上升子序列



## 状态变量：

`f[i]` 记录**以 `a[i]` 为结尾**的最长上升子序列的长度；

初始条件： `f[i] = 1`

递推式：

```cpp
从前往后
if(a[j] < a[i] && f[j]+1 < f[i]){
    f[i] = f[j] + 1;
}
```



也可以`f[i]` 记录**以 `a[i]` 为开头**的最长上升子序列的长度；



## 代码 $O(n^2)$

`f[i]` 记录**以 `a[i]` 为结尾**的最长上升子序列的长度；

```cpp
#include <iostream>
using namespace std;
const int N = 1e5+5;
int dp[N],a[N];
int n,ans;
int main(){
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    for(int i=1;i<=n;i++)dp[i] = 1;
    for(int i=2;i<=n;i++){
        for(int j=1;j<i;j++){
			if(a[j] < a[i] && f[j]+1 < f[i]){
    			dp[i] = max(dp[i],dp[j] + 1);
			}
        }
        ans = max(ans,dp[i]);
    }
    cout<<ans;
    return 0;
}
```





# 二分优化版本

如果把内层循环改为二分查找，就能把内层查找时间降为 $O(logn)$，时间复杂度降为 $O(nlogn)$。

但是，二分查找的前提是有序序列，故增加一个b数组，用来记录上升子序列。

**关键问题：动态更新b数组**



考虑新进来一个元素 `a[i]`：

1. **大则添加**：如果 `a[i]` 大 于 `b[len]` ，直接让 `b[++len]=a[i]` 。即b数组的长度增加1，而且添加了一个元素；

2. **小则替换**：如果 `a[i]` 小于或等于 `b[len]`，就用`a[i]`替换掉b数组中第一个**大于或等于`a[i]`的元素**；

   假设第一个大于`a[i]`的元素是`b[j]`

​	那么用 `a[i]` 换掉 `b[j]` 后，会使得 `b[1...]` 这个上升子序列的结尾元素更小。对于一个上升子序列，**其结尾元素越小，越有利于续接其它元素**，也就越可能变得更长。



**注意：**只有长度和最长上升子序列的长度是一样的，而 `b[]` 数组储存的并一定不是真正的最长上升子序列。



## 代码 $O(nlogn)$

```cpp
#include <iostream>
using namespace std;
const int N = 1e5+5;
int dp[N],a[N],b[N],len;
int n,ans;
int find(int x){
    int l=0,r = len + 1;
    while(l+1<r){
        int mid = (l+r)>>1;
        if(b[mid]<x)l = mid;
        else r = mid;
    }
    return r;
}
int main(){
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    for(int i=1;i<=n;i++)dp[i] = 1;
    b[++len] = a[1];
    for(int i=2;i<=n;i++){
        if(b[len]<a[i]){
            b[++len] = a[i];
        }
        else {
            int j = find(a[i]);
            b[j] = a[i];
        }
    }
    cout<<len;
    return 0;
}
```
