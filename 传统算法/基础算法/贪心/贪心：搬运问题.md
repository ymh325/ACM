有 2 个数组 `a` , `b` ，长度相等且为 `n` ，$\sum{a_i} = \sum{b_i}$

把 `a` -> `b` 的**最小代价**为 $\sum{\left|\;pre(A_i)\;\right| - \left|\;pre(B_i) \;\right|}$


```cpp
int ans=0;
int a[N],b[N];
int pre_a[N,pre_b[N];
for(int i=1;i<=n;i++){
	pre_a[i] = a[i]+pre_a[i-1];
}
for(int i=1;i<=n;i++){
	pre_b[i] = b[i]+pre_b[i-1];
}
for(int i=1;i<=n;i++){
	ans += abs(pre_a[i]-pre_b[i]);
}

```
