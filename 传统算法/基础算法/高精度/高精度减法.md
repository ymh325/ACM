# **高精度减法的步骤：**

1. 高精度数字利用字符串读入；
2. 把字符串翻转存入两个整型数组 A,B；
3. 若 A<B，则交换 A,B，输出负号；
4. 从低位到高位，逐位求差，借位，存差；
5. 把数组 C 从高位到低位依次输出。

```cpp
const int N = 1e5+5;
int la,lb,lc;
int A[N],B[N],C[N];

int cmp(int A[],int B[]){
	if(la!=lb)return la>lb;
	for(int i=la-1;~i;i--){
		if(A[i]!=B[i])return A[i]>B[i];
	}
	return 1; // 避免结果为 -0
}
void sub(int A[],int B[],int C[]){
	for(int i=0;i<lc;i++){
		if(A[i]<B[i]){
			A[i+1]--,A[i]+=10;
		}
		C[i] = A[i]-B[i];
	}
	while(lc&&C[lc]==0)lc--;  // 处理前导0
}
int main(){
	string a,b;
	la = a.size(),lb = b.size();
	lc = max(la,lb);
	for(int i=la-1;~i;i--)A[la-1-i] = a[i]-'0';
	for(int i=lb-1;~i;i--)B[lb-1-i] = b[i]-'0';
	if(!cmp(A,B))swap(A,B),cout<<"-";
	sub(A,B,C);
	for(int i=lc;~i;i--)cout<<C[i];
	return 0;
}
```
