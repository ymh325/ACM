> [!done] ST表
> 主要用来解决 RMQ（区间最大/最小值查询）问题。主要应用于 **倍增思想**，可以实现 $O(nlog{n})$ 预处理，$O(1)$查询


# 预处理ST表

倍增递推：用两个等长小区间拼凑一个大区间；

$f[i][j]$ 以第 $i$ 个数为起点，长度为 $2^{j}$ 的区间中的最大值； 

![[ST表.png]]

$$
f[i][j] = max(f[i][j-1],f[i+2^{j-1}][j-1])
$$

```cpp
for(int i=1;i<=n;i++) cin<<f[i][0];
for(int j=1;j<=20;j++){ // 枚举区间长度
	for(int i=1;i+(1<<j)-1<=n;i++){ // 枚举区间起点
		f[i][j] = max(f[i][j-1],f[i+(1<<(j-1))][j-1]);
	}
}
```

区间终点：$i+2^j-1 \le n$


# 处理查询

对查询区间 $[l,r]$ 做分割、拼凑
区间长度的指数 $k = \lfloor log_2{(r-l+1)} \rfloor$

|  K  |      区间长度       |
| :-: | :-------------: |
|  0  |       {1}       |
|  1  |      {2，3}      |
|  2  |    {4，5，6，7}    |
|  3  | {8，9，10，...，15} |

$\because k\;是向下取整得到$
$\therefore 2^k \leq r-l+1$，且 $r-l+1 \leq 2^{k+1}$ 

$故,2^k \leq r-l+1 \leq 2^{k+1}$

即区间 $[l,r]$ 必可以用两个长度为 $2^k$ 的区间 **重叠拼凑**

![[ST重叠查询.png]]
即 $max(f[l][k],f[r-2^k+1][k])$

设有 $q$ 个查询
```cpp
while(q--){
	int l,r;cin>>l>>r;
	int k = log2(r-l+1);
	int mx = max(f[l][k],f[r-(1<<k)+1][k]);
	cout<<mx<<'\n';
}
```


# 尾言

> [!important] 使用
> 凡是符合**结合律**且**可重复贡献**的信息**查询**都可以使用ST表。
> 显然最大值、最小值、最大公因数、最小公倍数、按位或、
> 按位与都符合这个条件。
> 如果涉及区间修改操作，就要使用**线段树**啦。
>
