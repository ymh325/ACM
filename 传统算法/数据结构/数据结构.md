# 状态维护

- 维护顺序关系
	- 数组、链表、栈、队列

- 维护最值
	- 堆、单调栈、单调队列

- 维护连通性
	- 并查集

- 维护层次关系
	- 树、Trie、平衡树

- 维护区间信息
	- 树状数组、线段树

- 维护映射关系
	- 哈希表


# 学习数据结构的方法论

- 先看数据结构要维护什么信息
	- 值
	- 下标
	- 排名
	- 区间
	- 连通块

- 再看支持哪些操作
	- 插入
	- 删除
	- 修改
	- 查询
	- 合并

- 再看复杂度
	- 单次操作复杂度
	- 建结构复杂度
	- 空间复杂度

- 再看实现方式
	- 数组模拟
	- 指针 / 引用
	- 递归
	- 迭代

- 最后看适用场景
	- 静态问题还是动态问题
	- 单点操作还是区间操作
	- 离线还是在线


# 刷题时需要想什么

- 为什么要用这个数据结构
- 它维护的核心信息是什么
- 每个操作如何实现
- 能否做到更优复杂度
- 是否需要懒标记、路径压缩、按秩合并、动态开点等优化
