# 对顶堆

## 定义


> [!done] 简述
> 
**对顶堆**有一个大根堆与一个小根堆组成，小根堆维护前 $K$ 大的数（包含第 $K$ 个），大根堆维护比第 $K$ 大数小的数


## 作用

可以动态维护 一个序列上第K大的数，K值会发生变化。比写 **线段树** 或 **BST** 简单

## 操作

1. **插入**：若插入的元素 ≥ 小根堆堆顶元素，则将其插入小根堆，否则将其插入大根堆。

2. **维护**：

   - 当小根堆的大小>k时，不断将小根堆堆顶元素取出并插入大根堆，直到小根堆的大小等于k;

   - 当小根堆的大小<k时，不断将大根堆堆顶元素取出并插入小根堆，直到小根堆的大小等于k。

1. **查询第k大元素：** 小根堆堆顶元素。
2. **删除第k大元素：** 删除小根堆堆顶元素。

## $Code$

时间复杂度：$O(nlog(n))$

```c++
#include<iostream>
#include<queue>
priority_queue<int>a;// 大根堆
priority_queue<int,vector<int>,greater<int>>b;// 小根堆

int main(){
    for(int i=1;i<=n;i++){
        //内部操作为log(n)
        int x;scanf("%d",&x);
        if(b.empty()||x>=b.top())b.push(x);//插入
        else a.push(x);
        
        while(b.size()>k)a.push(b.top()),b.pop();//调整
        while(b.size()<k)b.push(a.top()),a.pop();
        printf("%d",b.top());//取值 //b.pop;删除
    }
}
```
