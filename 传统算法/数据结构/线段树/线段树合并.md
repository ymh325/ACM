# 线段树合并



通常是 **动态开点** 的 **权值线段树** ，合并两颗线段树指的是 **合并两颗线段树的区间信息。**

这里我们以  **合并区间 `sum` ** 举例 （因为最好理解）



那么，假设我们现在已经合并到了两颗线段树 `x` , `y` 的`p` 节点。

1. 如果其中一个的 `p` 节点为空，那么返回另一个的 `p` 节点。

2. **如果已经合并到**两颗线段树的**叶子节点**，那么，就把 `y` 在 `p` 节点的值加到 `x` 上，并将 `p` 节点**返回**。 

3. **递归**处理**左子树**，**处理右子树**。

4. 用左右子树的值更新当前节点。

5. 将 `p` 节点返回。

 

`x`，`y` 传入的是两颗线段树的**根节点**

```c++
const int N = 1e5+5;
#define mid ((l+r)>>1)
int root[N];
int ls[N*50],rs[N*50],sum[N*50];
void pushup(int u){
	sum[u] = sum[ls[u]] + sum[rs[u]];
}
void merge(int x,int y,int l,int r){	
	if(!x||!y)return x+y; //* 如果其中一个为空，则返回另一个，全为空则返回0
    if(l==r){
        sum[x]+=sum[y];
        return x;
    }
    ls[x] = merge(ls[x],ls[y],l,mid);
    rs[x] = merge(rs[x],rs[y],mid+1,r);
    pushup(x);
 	return x;
}
```
