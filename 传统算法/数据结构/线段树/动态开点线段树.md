# 动态开点线段树


## 1、问题引入

在我们一般写线段树的时候，我们是以如下方式进行定义树结构的。

```c++
const int N = 1e5+5; // N为总线段区间范围
struct tree {
    int l,r,...;// 设置维护的值
}tr[N<<1];
```

可以观察到，`tr[N<<1]` 中 `N<<1` 是 `N` 的 4 倍。

- [ ] 那么我们设想一下，如果 `N` 很大，比如 `N = 1e9` 呢？
	是不是就会导致 定义`tr[N<<1]`的时候内存很大无法实现呢。

那么我们要想一个办法来解决这个问题。


## 2、解决思路

- **动态开点**

  其实我们不需要所有的节点，因为可能某些区间段并没有被修改过，如果我们在没有被修改过的区间开一`tree` 结构体进行维护的话，会导致内存浪费。
	**核心：** **动态开点，一边修改、查询，一边建树**


- **空间复杂度**

  设修改次数为 `m`，分裂开点的时候，最坏情况会两个分支，到叶子节点。

  所以单次修改的最坏开点数为 `2logn` ，所以，空间复杂度为 **$O(m*2logn)$**。

​	**估计内存：$m\ ×\ 2logn\ × \ 4\ ×\ 4 = 3\ ×\ 10^{5}\ ×\ 2log10^9\ ×16 = 3\ ×\ 10^5\ ×\ 60\ ×\ 16=288M> 250M$**  

​	不用考虑最坏，把 $60$ 变成 $50$ 就行。



## 3、代码

​	我们不再用 `u * 2` 而后 `u * 2 + 1` 代表左右儿子，而是用两个数组 `ls[u]` 和 `rs[u]` 记录左右两个儿子的编号。

以维护区间 `sum` 为例子

```c++
#include <iostream>
using namespace std;
#define OFF ios::sync_with_stdio(false);cout.tie(0);cin.tie(0);
#define int long long
const int N  = (3e5+5) * 50;// ! 3e5+5 为操作数
int n,q,root,tot;
int ls[N],rs[N];
int sum[N],tag[N];
void pushdown(int u,int l,int r){
    if(tag[u]){
        int k = tag[u];
        int mid = (l+r)>>1;
        if(!ls[u])ls[u] = tot++;
        if(!rs[u])rs[u] = tot++;
        sum[ls[u]] += k*(mid-l+1);
        sum[rs[u]] += k*(r-(mid+1)+1);
        tag[ls[u]] += k;
        tag[rs[u]] += k;
        tag[u]=0;
    }
}
void change(int &u ,int l,int r,int x,int y,int k){
    if(!u) u = ++tot;
    if(x<=l && r<=y){
        sum[u] += k*(r-l+1);
        tag[u] += k;
        return ;
    }
    pushdown(u,l,r);
    int mid = l + r >> 1;
    if(x <= mid) change(ls[u],l,mid,x,y,k);
    if(y > mid)change(rs[u],mid+1,r,x,y,k);
    sum[u] = sum[ls[u]] + sum[rs[u]];
}
int query(int &u,int l,int r,int x,int y){
    if(!u)return 0;
    if(x<=l&&r<=y){
        return sum[u];
    }
    pushdown(u,l,r);
    int mid = (l+r)>>1;
    int res = 0;
    if(x <= mid)res+=query(ls[u],l,mid,x,y);
    if(y > mid)res+=query(rs[u],mid+1,r,x,y);
    return res;
}
signed main(){
    OFF
    cin>>n>>q;
    for(int i=1;i<=n;i++){
        int x;cin>>x;
        change(root,1,n,i,i,x);
    }
    while(q--){
        int opt,x,y,k;cin>>opt>>x>>y;
        if(opt==1){
            cin>>k;
            change(root,1,n,x,y,k);
        }else {
            cout<<query(root,1,n,x,y)<<endl;
        }
    }
    return 0;
}
```
