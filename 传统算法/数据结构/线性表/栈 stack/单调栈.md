## 单调栈

使用栈，维护一个固定的变长窗口 `[1,i]` 内的单调序列，从栈顶取最值，进行计算或转移。

1. 每个元素从栈顶入栈，淘汰的元素从栈顶出栈
2. 栈顶是当前窗口最值



## 数组模拟单调栈

1. 用 `q[]` 模拟单调栈
2. 用一个指针 `top` 来栈口

定义函数 `f(i)` 代表数列中第 `i` 个元素之后第一个大于 $a_{i}$ 的元素的下标 。若不存在，则 `f(i) = 0`。

```cpp
const int N = 1e5+5;
int top=0,n;
int a[N],ans[N],q[N];
using namespace std;
int main(){
	scanf("%d",&n);
	for(int i=1;i<=n;i++)scanf("%d",&a[i]);
	for(int i=1;i<=n;i++){
        while(top>0 && a[q[top]]<a[i]){
            ans[q[top]] = i;
            top--;
        }
        q[++top] = i;
    }
    for(int i=1;i<=n;i++)printf("%d ",ans[i]);
	return 0;
}
```
