# 单调队列

> [!done] 简介
> 
描述：一个长为 $n$ 的序列 $a$ ，一个大小为 $k$ 的窗口，现在从左向右滑动窗口，每次滑动一个单位，求每次滑动后窗口中的 $max$ 和 $min$


**单调队列：** 队尾进队出队，队头出队（维护子队列的单调性）


1. **队尾出队**的条件：队列不空且新元素更优，队中旧元素队尾出队
2. 每个元素必然从**队尾出队**一次
3. **队头出队**的条件：队头元素滑出了窗口

注意：队列中存储元素的**下标**，方便判断队头出队



# 实现
## deque 

$deque$ 是一种双端对列。与普通的队列不同，头部和尾部均可以进行 **出队入队** 操作

- **`push_back()`**：在`deque`的尾部插入一个元素。
- **`push_front()`**：在`deque`的头部插入一个元素。
- **`pop_back()`**：删除`deque`尾部的元素。
- **`pop_front()`**：删除`deque`头部的元素。
- `back()`：访问尾部元素
- `front()`：访问头部元素

### 代码

以下为维护 $min$ 为例

```c++
#include <iostream>
#include <queue>
using namespace std;
deque<int> q;
const int N =1e5+5;
int a[N];
int n,k;
int main(){
    scanf("%d%d",&n,&k);
    for(int i=1;i<=n;i++)scanf("%d",&a[i]);
    for(int i=1;i<=n;i++){
        while(q.size() && a[q.back()]>=a[i])q.pop_back(); 
        q.push_back(i);     
        if(q.front()<i-k+1)q.pop_front();  
        if(i>=k)printf("%d ",a[q.front()]); 
    }
	return 0;
}
```

若是维护 $max$ 则将 ` a[q.back()]>=a[i]` 换成 ` a[q.back()]<=a[i]`



## 数组模拟

1. 新开一个数组 `q[]` 用于模拟单调队列
2. 用一个头指针 `h` 表示队头，`t` 表示队尾

### 代码

此处以维护 $min$ 为例

```c++
const int N =1e5+5;
int q[N],a[N];
int h=1,t=0;	//当前q[]中不含有元素
int n,k;
int main(){
    scanf("%d%d",&n,&k);
    for(int i=1;i<=n;i++)scanf("%d",&a[i]);
    for(int i=1;i<=n;i++){
        while(h<=t && a[q[t]]>=a[i])t--; //如果队列不为空，且新进来的元素更优，则旧元素出队
        q[++t]=i;     //新元素入队
        if(q[h]<i-k+1)h++;  //如果队头元素已经不在窗口范围内则出队
        if(i>=k)printf("%d ",a[q[h]]); //当窗口已经集齐元素时，开始输出元素
    }
}
```

若是维护 $max$ 则将 `a[q[t]] >= a[i]` 换成 `a[q[t]] <= a[i]`
