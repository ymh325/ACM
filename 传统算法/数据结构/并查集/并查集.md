# 并查集

> 一种树形的数据结构


**有两种操作**：

- **合并**：将两个子集合并成一个集合
- **查找**：确定某个元素处在哪个集合

`fa[x]`存节点 `x` 的父节点

`fa[]` 初始化：

每个节点是一个集合，每个节点的父亲是它自己

```c++
for(int i=1;i<=n;i++)fa[i]=i;
```

## 查找

> 查找元素所在集合的根

- 如果父节点等于自己，则找到了根并返回
- 如果还没找到根，则继续递归查找

```c++
int find(int x){
    if(fa[x]==x)return x;
    return find(fa[x]);
}
```

## 带路径压缩的查找

- 在返回的路上，顺带修改各节点的父节点为根，也就是让所有子节点指向最开头的根节点

```c++
int find(int x){
    if(fa[x]==x)return x;
    return fa[x]=find(fa[x]);
}
```

## 合并

> 把一个集合的根指向另一个集合的根

```c++
void merge(int x,int y){
    fa[find(x)]=find(y);
}
```
