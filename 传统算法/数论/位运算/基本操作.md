### 一、位运算



#### **🤔注意：**

1. **对于有符号数，补码中的符号位同样参与`&` `|` `^`运算**
2. **在原码$\rightarrow$ 反码时，除符号位其他位置取反；在按位~运算时，所有位置都取反**



#### 按位与`(&)`

对两个操作数的对应位进行逻辑与运算，只有当两个位都为 `1` 时结果才是 `1`，否则为 `0`。

```C++
int a = 5;      	// 二进制表示：0000 0101 (补码)
int b = 3;      	// 二进制表示：0000 0011 (补码)
int result = a & b;//  二进制结果：0000 0001 (补码)，十进制值为1
```



#### 按位或`(|)`

对两个操作数的对应位进行逻辑或运算，只要有一个位为 `1` 结果就是 `1`，只有两个位都为 0 时结果才为 `0`。

```C++
int a = 5;      	 // 二进制：0000 0101 (补码)
int b = 3;      	 // 二进制：0000 0011 (补码)
int result = a | b;//二进制结果：0000 0111 (补码)，十进制值为7
```



#### 按位异或`(^)`

对两个操作数的对应位进行逻辑异或运算，当两个位不同时结果为 `1`，相同时结果为 `0`。

```C++
int a = 5;      	 // 二进制：0000 0101 (补码)
int b = 3;      	 // 二进制：0000 0011 (补码)
int result = a ^ b;//二进制结果：0000 0110 (补码)，十进制值为6
```



#### 按位取反`(~)`

对操作数的每一位进行逻辑取反，即 `1` 变为 `0`，`0` 变为 `1`。

```C++
int a = 5;        //  二进制：0000 0101 (补码)
int result = ~a;// 二进制结果：1111 1010 (补码)，十进制值为-6
```



#### 左移`(<<)`

将左操作数的二进制位向左移动右操作数指定的位数，右边空出的位用 0 填充。左移 n 位相当于乘以 2 的 n 次方。

```C++
int a = 5;      // 二进制：0101
int result = a << 2;  // 二进制结果：010100，十进制值为20
```

**对于有符号数：**

1. **正数：**

   $26<<1=52$   ，$0001\;1010(补码) \xrightarrow{\text{左移1位}}0011\;0100(补码)=52 $

   $26<<2=104$ ，$0001\;1010(补码) \xrightarrow{\text{左移2位}}0110\;1000(补码)=104 $

2. **负数：**

​	$-26<<1=-52$，$1110\;0110(补码) \xrightarrow{左移1位} 1100\;1100(补码)\longrightarrow 1011\;0100(原码)=-52$



**故：$m<<n=m\cdot 2^n$**



#### 右移`(>>)`

把左操作数的二进制位向右移动右操作数指定的位数。对于无符号数，左边空出的位用 0 填充；对于有符号数，左边空出的位用符号位填充（算术右移）或者用 0 填充（逻辑右移），具体取决于编译器。右移 n 位相当于除以 2 的 n 次方。

```C++
int a = 5;      // 二进制：0101
int result = a << 2;  // 二进制结果：010100，十进制值为20
```

**对于有符号数：**

1. **正数右移：左侧补0**

   $26>>1=13$   ，$0001\;1010(补码) \xrightarrow{\text{右移1位}}0000\;1101(补码)=13 $

   $13>>1=6$     ，$0000\;1101(补码) \xrightarrow{右移一位} 0000\;0110(补码)=6$

2. **负数右移：左侧补1**

   $-26>>1=-13$， $1110\;0110(补码)\xrightarrow{右移1位}1111\;0011(补码)=-13$

   $-13>>1=-7$     ，$1000\;1101(补码) \xrightarrow{右移一位} 1100\;0110(补码)=-7$



**故：$m>>n=[m÷2^n]$**



#### 无符号右移`(>>>)`

1. **正数右移：左侧补0**
2. **负数右移：左侧补0**
