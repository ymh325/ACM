# 学习数论的方法论

- 先看题目在问哪一种整数关系
	- 整除
	- 同余
	- 质因数
	- 组合计数
	- 二进制状态

- 再看数据范围
	- 单次计算还是多次询问
	- n 是否能开数组预处理
	- 乘法是否会爆 long long
	- 模数是否为质数

- 再看需要什么预处理
	- 筛质数
	- 预处理阶乘和逆元
	- 预处理最小质因子
	- 预处理幂或前缀贡献

- 再看公式是否能转化
	- 除法转逆元
	- 指数降复杂度
	- 质因数分解转乘法原理
	- 同余关系转最短路或 DP

- 最后看边界
	- 0 和 1 的特殊性
	- 负数取模
	- 大数乘法溢出
	- 不互质时逆元不存在


# 刷题时需要想什么

- 题目核心是整除、同余、质数、组合数还是位状态？
- 是否有大量询问，需要预处理？
- 模数是不是质数，能不能使用费马小定理求逆元？
- 除法在模意义下是否合法？
- gcd 是否可以刻画可达性或最优性？
- 能否把数拆成质因数后独立处理？
- 是否需要用筛法优化枚举因子或倍数？
- 位运算中的状态是否合法，是否需要枚举子集 / 超集？
