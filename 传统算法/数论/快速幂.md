# 场景


> [!question] 取余运算
> 给定 3 个整数，$a$，$b$，$c$ ，求 $a^b\mod p$
> $0<=a,b<2^{31}$ ，$a+b>0$ ，$2<=p<2^{31}$ 


对于这个数据范围，我们可以知道如下的事情：
- 循环 $b$ 次相乘会有超时风险，时间复杂度为 $O(n)$；

# 快速幂


> [!success] 快速幂的快
>**快速幂** 使用 **二进制拆分和倍增思想**，仅需要 $O(\log{n})$ 


对 $n$ 做二进制拆分，以 $3^{13}$ 为例：
$$
3^{13} = 3^{(1101)_2} = 3^8 \cdot 3^4 \cdot 3^1
$$

**快速幂**可以应用在任何具有**结合律**的运算中；
例如：
$$
(3^{13})\%p = (\;(3^8)\%p \cdot (3^4)\%p \cdot (3^1)\%p\;)\; \% p
$$


## 无取模
```cpp
int quickpow(ll a,int n){
	int res=1;
	while(n){
		if(n&1)res*=a; // 如果当前位是1，就把对应的数乘上去
		a*=a; // 更新
		n>>=1;
	}
	return res;
}
```


## 取模
```cpp
int quickpow(ll a,int n,int p){
	int res=1;
	while(n){
		if(n&1)res= (res*a)%p; 
		a = (a*a)%p;  
		n>>=1;
	}
	return res;
}
```
