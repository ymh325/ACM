# 试除法判定质数

暴力做法，枚举从小到大的每个数看是否能整除，**时间复杂度为 $O(n)$** 。

事实上，如果 $a$ 是 $n$ 的约数，那么 $\frac {n}{a}$ 也是 $n$ 的约数。
对于每一对 $(a,\frac {n}{a})$ ，只需要检验其中小的那个数，所有较小的数就在这个区间 $[\;1,\sqrt{n}\;]$。
所以 **时间复杂度为 $O(\sqrt{n})$** 。 


# CODE $O(\sqrt{n})$

```cpp
bool is_prim(int x){
	if(x<=1)return 0;
	for(int i=2;i*i<=x;i++){
		if(x%i==0)return 0;
	}
	return 1;
}
```
