# $O(n)$

# 线性筛素数

>又称：欧拉筛法

## 理论

在之前的 [[埃氏筛法]] 中，我们会发现同一个合数可能杯标记了多次，
例如：$2\;×\;3=6$ ，在 $2$ 时，$6$ 被标记了一次；在 $3$ 时，$6$ 又被标记了一次。

所以，我们可以试试能不能**每个合数只被标记一次**？
就下来需要引入一个定理：

> [!abstract]  [[质因数分解定理|整数唯一分解定理]]
> 任何大于 1 的正整数 n 都可以**唯一**分解为有限个**质数**的乘积
> 任何合数都有他对应的**一个最小质因子**


## 流程

从小到大 **枚举每个数**
1. 如果当前数没有被划掉，必定是质数，记录该质数
2. **枚举已记录的质数** （如果合数以越界则中断）
	（1）合数未越界，则划掉合数
	（2）条件 `i%p==0`，保证合数 **只被最小质因子** 划掉
		若 $i$ 是质数，则最多枚举到自身中断；
		若 $i$ 是合数，则最多枚举到自身的最小质数中断；

## Code

```cpp
#define long long int
const int N = 1e5+5;
vector<int> prim;
vector<int> vis(N,1); // 真为质数
void find(int n){
	for(int i=1;i<=n;i++){
		if(vis[i])prim.push_back(i);
		for(int j=0;i*prim[j]<=n;j++){
			vis[i*prim[j]] = 0;
			if(i%prim[j]==0)break;
		}
	}
}

```

# Math illustation

设当前外层枚举到整数 $i$，内层依次枚举已经得到的质数 $p$，并标记合数
$$
x=i\cdot p.
$$

线性筛的核心是：**每个合数只在它的最小质因子那一次被标记**。

记 $\operatorname{spf}(x)$ 为 $x$ 的最小质因子。对于任意合数 $x$，可唯一写成
$$
x=i\cdot p,\quad p=\operatorname{spf}(x),\quad i=\frac{x}{p}.
$$

当外层走到这个 $i$ 时，内层一定会走到质数 $p$，于是 $x$ 被标记一次。


**为什么遇到 `i % p_j == 0` 要退出？**

假设当前质数是 $p$，且 $p$ 能整除 $i$（即 $i=p \cdot t$）。那对于任意比 $p$ 更大的质数 $q$，数

$$
i\times q=p \cdot t \cdot q
$$ 
的**最小质因子一定是 $p$**（因为 $p<q$ 且 $p$ 已经整除它）。

而我们的筛法要求“用合数的最小质因子来那一次标记它”，也就是这个数应该由 $t\cdot p$  与 $p$ 的乘积那次来标记（当处理 `i = t*q` 时，乘上 $p$ 会得到它）。

如果在 $i$ 的循环里继续用更大的 $q$ 去标记，就会造成 “同一个合数被不同方式标记多次”。

因此一旦发现 $p$ 整除 $i$，就立刻 `break`：停止用更大的质数去乘这个 $i$；
保证每个合数只由“以其最小质因子”那一次被标记到，从而达到线性复杂度。

也正因为每个合数只被写入一次，整体标记次数与 $1\sim n$ 内合数个数同阶，所以时间复杂度是
$$
O(n).
$$
