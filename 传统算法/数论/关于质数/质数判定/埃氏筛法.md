[[素数（质数）|什么是质数]]

# $O(nlog{log{n}})$

# 埃氏筛法

从小到大 **枚举每个数**
	如果当前数没划掉，必定是质数，记录该质数
		**枚举当前质数的倍数**，必定是合数，划掉合数


# Code 

```cpp
#define long long int 
const int N = 1e5+5;
vector<int> isprim(N,1);

void check(int n){
	for(int i=2;i*i<=n;i++){
		if(isprim[i]){
			for(int j=i*i;j<=n;j+=i){
				isprim[i] = 0;
			}
		}
	}
} 
```


# 时间复杂度证明



埃氏筛法的时间复杂度为 $O(n \log \log n)$。下面是其数学推导与证明：

1. **内层循环执行次数**：对于选定的一个质数 $p$，我们需要在 $n$ 的范围内划掉它的所有倍数（即 $2p, 3p, ...$），操作次数约为 $\frac{n}{p}$ 次。
2. **总操作次数**：将所有小于等于 $n$ 的质数的内层操作次数相加，总操作次数 $T(n)$ 可以表示为：
   $$ T(n) = \sum_{p \le n, p \in Prime} \frac{n}{p} = n \sum_{p \le n, p \in Prime} \frac{1}{p} $$
3. **质数倒数和定理**：根据欧拉（Euler）的发现以及梅滕斯第二定理（Mertens' 2nd theorem），所有小于等于 $n$ 的质数的倒数之和可以用对数积分近似，其渐进趋近于：
   $$ \sum_{p \le n, p \in Prime} \frac{1}{p} \approx \ln(\ln n) + M $$
   其中 $M$ 是梅滕斯常数（近似值为 $0.261497\dots$）。
4. **得出结论**：将上式代入总操作次数中，忽略常数部分，可以得出总的时间复杂度趋近于：
   $$ T(n) \approx n \ln(\ln n) $$

综上所述，埃氏筛法的最终代码时间复杂度为 $O(n \log \log n)$。
