# 组合数

## 定义

从 $n$ 个**互不相同**的元素中，任意取出 $k$ ($0 \le k \le n$) 个元素组成一个子集（不考虑元素的排列顺序），这种元素的组合总数被称为**从 $n$ 中取 $k$ 的组合数**。

在数学符号中，通常记作 $C_n^k$、$C(n, k)$ 或最常用的二项式系数符号 $\binom{n}{k}$。

>简单来说，**组合数**就是从 $n$ 个不同的东西里，不分先后顺序地挑出 $k$ 个东西的方法总数。


## 计算公式

组合数可以通过 **阶乘** 来定义：

$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

其中：

- $n! = n \times (n-1) \times \dots \times 1$
    
- 规定 $0! = 1$


**直观推导：**

我们可以先求出排列数 $A_n^k$（考虑顺序），然后除以这 $k$ 个元素自身的全排列 $k!$（消除顺序影响）：

$$\binom{n}{k} = \frac{A_n^k}{k!} = \frac{n(n-1)\dots(n-k+1)}{k!}$$

## 递推式

在 **算法程序中**，可以用递推式来进行推导，如下。

$$
C_n^m = C_{n-1}^m+C_{n-1}^{m-1}
$$

从 $n$ 个不同的数中选出 $m$ 个方案数是 $C_n^m$  ，
对第 $1$ 个数有 **选或不选** 两种决策：
- 若不选，则从剩下的 $n-1$ 个中选 $m$ 个，即 $C_{n-1}^m$；
- 若选，则从剩下的 $n-1$ 个中选 $m-1$ 个，即 $C_{n-1}^{m-1}$。

# 杨辉三角公式

- $C_n^0 = C_n^n=1$
	
- $C_n^m = C_n^{n-m}$ 
	
- $C_n^m = C_{n-1}^m + C_{n-1}^{m-1}$

# Code 

## 双层循环 

Time： $O(n^2)$

可以用 $C[i][j]$ 表示 $C_i^j$ ，即：
$$
C[i][j] = C_i^j
$$

```cpp
for(int i=0;i<=n;i++){
	for(int j=0;j<=i;j++){
		if(j==0) C[i][j] = 1;
		else C[i][j] = C[i-1][j]+C[i-1][j-1];
	}
}
```
