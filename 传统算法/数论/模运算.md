# 模运算


> [!important] YOU CAN KNOW
> 在一类题中，要求对答案取模。那么模运算的正确写法显得格外重要。

# 没有负数

​	当过程中不存在负数的出现时，这个情况很好处理。有以下公式。

​				

​				`(a + b) % mod = (a % mod + b % mod) % mod`

​				`(a * b) % mod = (a % mod * b % mod) % mod`



# 有负数

​	当过程中有负数的出现时，我们需要转换一下。

无论 a、b 是正还是负，都可以用以下统一流程，确保结果正确：

1. 对 a 做非负转换：`a_mod = (a % mod + mod) % mod`
2. 对 b 做非负转换：`b_mod = (b % mod + mod) % mod`
3. 加法模：`(a_mod + b_mod) % mod`
4. 乘法模：`(a_mod * b_mod) % mod`
