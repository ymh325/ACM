# 各种方式 

## 1、邻接表

**出边数组** `e[u][i]` 存储 `u` 点的所有出边的 `{终点，边权 w}`

- **时间复杂度：** $O(n + m)$
- **空间复杂度：** $O(n + m)$

**n** 代表**顶点数量**

**m** 代表**边的数量**



**应用：** 各种图，不能处理反向边



$CODE$

```c++
struct edge{ int v,w;};
vector<edge> e[N];// 存储每个顶点的边集合

void dfs(int u,int fa){
    for(auto ed:e[u]){
        if(ed.v==fa)continue;
        int v=ed.v,w=ed.w;
        //printf();
        if(v==fa)continue;
        dfs(v,u);
    }
}
int main(){
    cin>>n>>m;
    for(int i=1;i<=m;i++){
    	cin>>a>>b>>c;
        e[a].push_back({b,c});
        e[b].push_back({a,c});
    }
    dfs(1,0);
    return 0;
}

```





## 2、链式邻接表

**边集数组**  `e[]` 存储第 `j` 条边的 `{起点u,终点v,边权w}`

**表头数组** `h[u]` 存储 `u` 点的所有出边的编号

- 时间复杂度：$O(n+m)$

- 空间复杂度：$O(n+m)$

应用：各种图，能处理反向边



$CODE$

```C++
struct edge{int u,v,w;};
vector<edge> e; // 边集
vector<int> h[N]; // h[i]为 i 节点的所有出边编号的集合 
void dfs(int u,int fa){
    for(int ed:h[u]){
        int v=e[ed].v,w=e[ed].w;
        if(v==fa)continue;
        dfs(v,u);
    }
}
void add(int a,int b,int c){
    e.push_back({a,b,c});
    h[a].push_back(e.size()-1);
}
int main(){
    cin>>n>>m;
    for(int i=1;i<=m;i++){
        cin>>a>>b>>c;
        add(a,b,c);
        add(b,a,c);
    }
    dfs(1,0);
    return 0;
}
```



## 3、链式前向星

一个表头数组悬挂多个链表
**边集数组** `e[i]` 存储第 `i` 条出边的 `{终点v,边权w，下一条边ne}`
**表头数组** `h[u]` 存储 `u` 点的第一条出边的编号
**边的编号**  `idx`  可取 $0,1,2,3...$

- 时间复杂度：$O(n+m)$

- 空间复杂度：$O(n+m)$

应用：各种图，能处理反向边

$CODE$

```C++
struct edge{int v,w,ne;};
edge e[M]; // 边集
int idx,h[N]; // 点的第一条出边
void dfs(int u,int fa){
    for(int i=h[u];~i;i=e[i].ne){
        int v=e[i].v,w=e[i].w;
        if(v==fa)continue;
        dfs(v,u);
    }
}
void add(int a,int b,int c){
    e[idx]={b,c,h[a]};
    h[a]=idx++;
}
int main(){
    cin>>n>>m;
    memset(h,-1,sizeof h);
    for(int i=1;i<=m;i++){
        cin>>a>>b>>c;
        add(a,b,c);
        add(b,a,c);
    }
    dfs(1,0);
    return 0;
}
```
