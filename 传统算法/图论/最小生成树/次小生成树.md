# 次小生成树

​	先回顾一下最小生成树的概念：

- 最小生成树：在一个带边权的无向连通图中，找到一颗 **连接所有顶点**，并且**所有边的权值之和最小**的子树



## 非严格次小生成树



### 定义

在无向图中，边权和最小的满足边权和 **大于等于** 最小生成树边权和的生成树

### 求解方法

- 求出无向图的最小生成树 𝑇 ，设其权值和为 𝑀
- **遍历每条未被选中的边 𝑒 =(𝑢,𝑣,𝑤)**，找到 𝑇 中 𝑢 到 𝑣 路径上边权最大的一条边 𝑒′ =(𝑠,𝑡,𝑤′)，则在 𝑇 中以 𝑒 替换 𝑒′，可得一棵权值和为 𝑀′ =𝑀 +𝑤 −𝑤′ 的生成树 𝑇′.
- 对所有替换得到的答案 𝑀′ 取最小值即可

如何求 𝑢,𝑣 路径上的边权最大值呢？

我们可以使用倍增来维护，预处理出每个节点的 2𝑖 级祖先及到达其 2𝑖 级祖先路径上最大的边权，这样在倍增求 LCA 的过程中可以直接求得。



## 严格次小生成树

### 定义

在无向图中，边权和最小的满足边权和 **严格大于** 最小生成树边权和的生成树

### 求解方法

考虑刚才的非严格次小生成树求解过程，为什么求得的解是非严格的？

​	因为在执行换边操作时，**新边和旧边的权值可能相同**。换言之，当我们用于替换的边的权值与原生成树中被替换边的权值相等时，**得到的次小生成树是非严格的。**

​	那我们应该如何解决呢？

​		**解决的办法很自然：**我们维护到 **2𝑖**  级祖先路径上的最大边权的同时维护 **严格次大边权**，当用于替换的		边的权值与原生成树中路径最大边权相等时，我们用严格次大值来替换即可。



## 代码

如下时求严格次小生成树边权的示例

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;
#define OFF ios::sync_with_stdio(false),cin.tie(0), cout.tie(0);
#define int long long
const int N = 1e5 + 5;
const int M = 3e5 + 5;
const int  inf = 1e18;
struct edge {
    int u,v,w;
    bool operator < (const edge& p)const {
        return w< p.w;
    }
}e[M];
struct node{int v,w;};
vector<node> G[N];
int n,m,ans;
int fa[N];
int st[N][20],mx[N][20],mi[N][20],dep[N],vis[N];
int find(int x){
    if(fa[x] == x) return x ;
    return fa[x] = find(fa[x]);
}
void krs(){
    for(int i=1;i<=n;i++)fa[i] = i;
    sort(e+1,e+m+1);
    for(int i=1;i<=m;i++){
        int x = e[i].u,y = e[i].v,w = e[i].w;
        int u = find(e[i].u);
        int v = find(e[i].v);
        if(u!=v){
            fa[u] = v;
            vis[i] = 1;
            ans += w; 
            G[x].push_back({y,w});
            G[y].push_back({x,w});
        }
    }
}
void get_st(int u,int fat,int w){
    st[u][0] = fat;
    mx[u][0] = w;
    mi[u][0] = -inf;
    dep[u] = dep[fat] + 1;
    for(int i=1;i<=19;i++){
        st[u][i] = st[st[u][i-1]][i-1];
        
        int t[4] = {
            mx[u][i-1],mx[st[u][i-1]][i-1],
            mi[u][i-1],mi[st[u][i-1]][i-1]
        };
        sort(t,t+4);
        mx[u][i] = t[3];

        int pos = 2;
        while(pos>=0 && t[pos] == t[3])pos--;
        mi[u][i] = (pos == -1 ? -inf:t[pos]); 
    }
    for(node ed:G[u]){
        if(ed.v != fat)get_st(ed.v, u , ed.w);
    }
}
int lca(int u,int v){
    if(dep[u]<dep[v])swap(u,v);
    for(int i=19;i>=0;i--){
        if(dep[st[u][i]] >= dep[v])u = st[u][i];
    }
    if(v==u)return u;
    for(int i=19;i>=0;i--){
        if(st[u][i] != st[v][i]){
            u = st[u][i],v = st[v][i];
        }
    }
    return st[u][0];
}
int query(int u,int lca,int val){
    int res=-inf;
    if(u == lca)return res;
    for(int i=19;i>=0;i--){
        if(dep[st[u][i]] >= dep[lca]){
            if(val != mx[u][i]){
                res = max(res,mx[u][i]);
            }
            else {
                res = max(res,mi[u][i]);
            } 
            u = st[u][i];
        }
    }
    return res;
}

signed main(){
    OFF
    cin>>n>>m;
    for(int i=1;i<=m;i++){
        int x,y,z;cin>>x>>y>>z;
        e[i] = {x,y,z};
    }
    krs();
    get_st(1,0,0);
    int res = inf;
    for(int i=1;i<=m;i++){
        if(vis[i] == 0){
            int u = e[i].u, v = e[i].v;
            int ca = lca(u,v);
            int l = query(u,ca,e[i].w);
            int r = query(v,ca,e[i].w);
            int bian = max(l,r);
            if(bian ==  -inf)continue;
            res = min(res,ans-bian +e[i].w);
        }
    }
    cout<<(res == inf ? -1:res); // 不存在则输出 -1
    return 0;
}
```



## 扩展

​	刚刚我们写了一个很完整的求解 **严格次小生成树的** 的代码方案，用 **倍增 + LCA** 可以很好地理解求出严格的于非严格的。

那么我们如何**更简单地**寻找一个**图中是否有多个最小生成树呢**？

### 思路 

​	首先我们想一想，要看是否有多个最小生成树，其实就是把最小生成树的边选完过后，再看未选择的边。如果某个未选择的边**可以替换掉相等权重**的已选边，那么就是有其他的最小生成树。我们可以用 **倍增 + LCA** 解决，但不优雅，我们可以有更简洁的方法。

我们要在一开始选边的时候，动一下手脚。
