# 基环树


> [!NOTE] 简介
> 一个图包含 `n` 个点 `n` 条边，且图中只存在一个环，这样的图被称为 **基环树**
>> 又称环套树

---

## 分类：
- 无向树
![[无向树.png|316]]

- 外向树（每个点只有一条出边）
![[外向树.png]]


- 内向树（每个点只有一条出边）
![[内向树.png]]


## 性质：


> [!important] 共性
> 所有种类的基环树，断掉环上的任意一条边后，都会退化为 **树形结构**


### 无向基环树

- 任意删去环上的一条边，会退化为普通的树，一颗 **无向树**

### 内向基环树

- 环外的节点像河流汇入大海一样，最终都指向环
- **入度：** >=0
- **出度**：严格 $= 1$

### 外向基环树

- 从环上的节点出发，可以到达对应的子树中的所有结点
- **入度：** 严格 $= 1$
- **出度：** >=0

## 基本思路：
1. 深搜找环
2. 断环成树，对树深搜计算





# 例题

[P2607 骑士 - 洛谷](https://www.luogu.com.cn/problem/P2607)

**题意：** 每个骑士有一个不可以同时上场的骑士，和一个战斗力。求最大战斗力
**转化：** `n` 个点 `n` 条边，每个点有权值，有边相连的两个点只能选其一，求可选方案的最大的点权之和。

**思路：**
1. 建图后，其实就是有多个基环树
2. 对于每颗基环树，先断开环上的任意一条边，对两个端点分别做一次树形DP，两次答案求最大值
3. 然后，累加每棵树的最大值即可；

树形DP类似于：[P1352 没有上司的舞会 - 洛谷](https://www.luogu.com.cn/problem/P1352)

```cpp
#include <bits/stdc++.h>
using namespace std;
#define OFF ios::sync_with_stdio(0),cin.tie(0); 
#define int long long
const int N = 1e6 + 5;

int n;
vector<int> e[N];
int vis[N],w[N];
int r1,r2;
void find(int u,int rt){ // 找两个根
    vis[u]=1;
    for(int v:e[u]){
        if(v==rt){
            r1=u,r2=v;
            return;
        }
        if(vis[v])continue;
        find(v,rt);
    }
}
int f[N][2];
int dfs(int u,int rt){ // 树上DP
    f[u][0]=0;f[u][1] = w[u];
    for(int v:e[u]){
        if(v==rt)continue;
        dfs(v,rt);
        f[u][0] += max(f[v][0],f[v][1]);
        f[u][1] += f[v][0];
    }
    return f[u][0];
}
signed main(){
    OFF
    cin>>n;
    for(int i=1;i<=n;i++){
        int v;
        cin>>w[i]>>v;
        e[v].push_back(i); // 单向边
    }
    int sum=0;
    for(int i=1;i<=n;i++){
        if(!vis[i]){
            r1=r2=0;
            find(i,i);
            if(r1){
                int res1 = dfs(r1,r1);
                int res2 = dfs(r2,r2);                
                sum += max(res1,res2);
            }
        }
    }
    cout<<sum;
    
    return 0;
}
```
