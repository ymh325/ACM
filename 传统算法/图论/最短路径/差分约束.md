# 负环和差分约束



**差分约束的两种形式**

**形式1：**给定若干的不等式，类似 $X_i-X_j < = C_i$ ，判断所有不等式是否有解，有解给出变量的一组解

其中 $X_i$ ，$X_j$ 均为变量，$C_i$ 为常量

**形式1：**给定若干的不等式，类似 $X_i-X_j > = C_i$ ，判断所有不等式是否有解，有解给出变量的一组解

其中 $X_i$ ，$X_j$ 均为变量，$C_i$ 为常量



形式1 和 形式2 可以相互转化，**形式1判断负环（最短路）**，**形式2判断无限增加的环（最长路）** 



## 形式1

我们可以转换一下，把$X_i$、$X_j$ 看作是图的节点， $C_i$ 看作两点之间的边权
$$
X_i-X_j < = C_i \quad => \quad U-V <= W \\
U-V <= W \quad => \quad U <= V + W
$$


每一个不等式都这样处理，可以转换成图的形式

**将每个不等式进行相互带换，对于任意在不等式中出现的节点，都会有如下公式：**
$$
U < = U + C
$$
那么，只要 任意在不等式中出现的节点的此公式 的常数项 $C$ 不为 负数

**即整个图中不存在负环，则不等式方程组存在可行解**

 



# 题目描述

> P5960 【模板】差分约束

给出一组包含 m 个不等式，有 n 个未知数的形如：

$\begin{cases} x_{c_1} - x_{c_1'} \leq y_1 \\ x_{c_2} - x_{c_2'} \leq y_2 \\ \cdots \\ x_{c_m} - x_{c_m'} \leq y_m \end{cases}$

的不等式组，求任意一组满足这个不等式组的解。

## 输入格式

- 第一行为两个正整数 $(n, m)$，代表未知数的数量和不等式的数量。
- 接下来 m 行，每行包含三个整数 $(c, c', y)$，代表一个不等式 $(x_c - x_{c'} \leq y)$。

## 输出格式

一行，`n` 个数，表示 $(x_1, x_2 \cdots x_n)$ 的一组可行解，如果有多组解，请输出任意一组，无解请输出 `NO`。



```cpp
#include <iostream>
#include <vector>
#include <cstring>
#include <queue>
#define OFF ios::sync_with_stdio(false); cin.tie(0); cout.tie(0);
using namespace std;
const int N = 5e3 + 5;
struct edge{int v,w;};
vector<edge> e[N];
queue<int> q;
int d[N],cnt[N],vis[N];
int n,m;
int spfa(int s){
    memset(d,0x3f,sizeof(d));
    d[s]=0;q.push(s);vis[s]=1;
    while(q.size()){
        int u = q.front();q.pop();
        vis[u] = 0; //* 标记不在队列中
        for(edge ed:e[u]){
            int v = ed.v ,w = ed.w;
            if(d[v]>d[u]+w){
                d[v] = d[u]+w;
                cnt[v] = cnt[u] + 1;
                if(cnt[v]>=n+1)return 1;//* 有负环，因为添加了一个超级源点，所以 >= n+1
                if(!vis[v])q.push(v),vis[v]=1;
            }
        }
    }
    return 0;//* 无负环
}
int main(){
    OFF
    cin>>n>>m;
    while(m--){
        int u,v,w;cin>>u>>v>>w;
        e[v].push_back({u,w});
    }
    for(int i=1;i<=n;i++){//* 超级源点,可到达所有点,判断整张图是否有负环
        e[0].push_back({i,0});
    }
    int op = ford(0);
    if(op)cout<<"NO";
    else {
        for(int i=1;i<=n;i++){
            cout<<d[i]<<" ";
        }
    }
    return 0;
}
```
