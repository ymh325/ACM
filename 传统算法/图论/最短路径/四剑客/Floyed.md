# Floyd

**弗洛伊德**算法 是 动态规划 算法，也称插点法。是 **全源最短路算法**

**状态表示：** `d[k,i,j]` 表示从 `i` 到 `j` ，且中间只经过节点编号 `1~k` 的最短路径的长度



**状态计算**

路径的选择**分两类**：

1. 路径经过 `k` 点，**继承原值：**`d[k,i,j] = d[k-1,i,j]`
2. 路径记过 `k` 点，**松弛操作：**`d[k,i,j] = d[k-1,i,j] + d[k-1,k,j]`

```c++
void floyd(){
    for(int k=1;k<=n;k++)
        for(int i=1;i<=n;i++)
                for(int j=1;j<=n;j++)
                    dis[i][j] = min(dis[i][j],dis[i][k]+dis[k][j]);
}
```





# 狭义与广义

> 以下均为自己的理解



## 关系的递进

- **狭义下**，`Floyd` 是来求 **图的全源最短路** 的

- **广义下**，`Floyd`  中的 `dis[i][j]` 是求 `i` 点是否可以到达 `j` 点的。
  - 进一步地，可以说是  `i` 和 `j` 存在着一种**可传递的关系**



那么模板下的 `Floyd` 是狭义的。

接下来，我们讨论广义的代码是怎样的：

- 如果 `dis[i][k] == 1` 且 `dis[k][j] == 1` ，则 `i` -> `k` ，`k` -> `j` ，故 `i` -> `j`
  - 举个例子，`i` 比 `k` 大，`k` 比 `j` 大，则 `i` 比 `j` 大。关系靠 `k` 为媒介进行传递



那么代码可以变成如下：

```c++
void floyd(){
    for(int k=1;k<=n;k++)
        for(int i=1;i<=n;i++)
                for(int j=1;j<=n;j++)
                    dis[i][j] |= dis[i][k] & dis[k][j];
}
```



## 对于循环的理解

我们先抬出模板。

```c++
void floyd(){
    for(int k=1;k<=n;k++)
        for(int i=1;i<=n;i++)
                for(int j=1;j<=n;j++)
                    dis[i][j] = min(dis[i][j],dis[i][k]+dis[k][j]);
}
```

现在我们需要讨论 最外层 `k` 循环的含义。

- 对于每一次循环，我们可看作是任意两点间的距离围绕着 `k` 去更新。以 `k` 为支点更新了
- `k` 可次循环可看作是 任意两点的路径 通过这 `1~k` 个点进行了松弛操作
