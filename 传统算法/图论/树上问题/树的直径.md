# 树的直径



## 定义

**直径定义**：树的直径是指该树中所有顶点对之间的路径长度的最大值

> 直径**不一定**是唯一的，但可以证明：各直径的**中点**（不一定恰好是某个结点，可能在某条边的内部）是**唯一**的



**有两种求解思路：**

1. **两次DFS：** **缺点**：仅适用于正边权，**优点**：方便记录直径的路径。时间复杂度：$O(n)$
2. **树形DP：** **缺点：** 不方便记录直径的路径，**优点：**适用于正负边权。时间复杂度：$O(n)$





## 解决思路 ①

首先从任意节点 `x` 开始进行第一次 `DFS`，到达距离其最远的节点 `y`

然后再从 `y` 开始做第二次 `DFS` ，到达距离 `y` 最远的节点 `z`，则  `[y,z]` 即为树的直径。

> 对于此的证明可以去往 OI Wiki 网站学习 
>
> [OI Wiki]: https://oi-wiki.org/graph/tree-diameter/



令 `dep[]` 数组存储节点的深度，`mxd` 为最大路径 ，`[y,z]` 为直径两端

此代码的图存储方式为 **邻接表**

```cpp
void dfs(int u, int fat){
    for(auto ed:e[u]){
        int v = ed.v, w = ed.w;
        if(v == fat)continue;
        dep[v] = dep[u] + w;
        if(dep[v] > mxd) mxd = dep[v],y=v;
        dfs(v,u);
    }
}
```

## 解决思路 ②

利用 `dfs` ，在搜索过程中顺带记录一些节点属性，从而求得树的直径

任取一点 `u` ，从 `u` 点向下搜，返回时收集边的权值，记录 **两条路径**

- **d1：** 记录从 `u` 点往下走的最长路径的长度 

- **d2：** 记录从 `u` 点往下走的次长路径的长度
  
- **对于存储树的直径长度，2种方式**
  1. **`d[u] = d1 + d2`**。表示悬挂在 `u` 点上的最长路径的长度，遍历取最大值即可
     
    2. 可以用全局变量 `ans` ，实时更新。**`ans = max(ans,d1+d2)`**
       



此代码的图存储方式为 **邻接表**

```cpp
int dfs(int u,int fat){  
     int d1 = 0,d2 = 0;  
     for(auto ed:e[u]){  
         int v = ed.v, w = ed.w;  
         if(v == fat)continue;  
         int d = dfs(v,u)+w;  
         if(d>d1)d2=d1,d1=d;  
         else if(d>d2) d2=d;  
     }  
     ans = max(ans,d1+d2);  
     return d1;  
 }
```
