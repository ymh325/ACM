# 树的重心



## 定义：

- **重心定义1：** 重心是指树中的一个结点，如果将这个点删除后，

​			    剩余各个连通块中节点数的最大值最小，那么这个结点被称为树的重心。

- **重心定义2：** 对于树中的某个节点，如果将它从树中删除后，

​			   剩下的所有连通块（子树）的节点数都不超过原树节点数的一半，那么这个节点就是树的重心。

> 注意：重心至少有一个，最多有两个。
>
> - **一个重心**：大多数情况，尤其是节点数为奇数时。
> - **两个重心**：当树具有对称性且节点数为偶数时发生，两个重心必是相邻节点。
> - **三个或以上**：不可能。



## 解决思路

​	利用 `dfs` ，在搜索过程中顺带记录一些节点属性，从而求得树的重心

任取一点 `u`，若以 `u` 为重心，则分为两类：

- **一类是u的子树**

- **一类是u上面的部分**

需要算出 `u` 的最大子树的结点数和 `u` 上面的部分的结点数，然后取二者的最大值即可。

- **size：** 记录 `u` 的最大子树的结点数；
	
- **sum：** 记录以 `u` 为根的子树的结点数；
	
- **n-sum：**  `u` 上面的部分的结点数;

  $ans = max(size,n-sum)$。



`size` 、`sum` **可以在向下遍历返回时统计和累加**

`vis[u]` ：标记`u` 这个点被搜索过


因为 `u` 是任取的一点，所以在遍历每个点时都会
得到一个 `ans`，取最小值即可。



## $CODE$

用邻接表即可

- 搜索

```cpp
int dfs(int u){
    vis[u] = true;
    int size = 0;
    int sum = 1;
    for(auto ed:e[u]){
        int j = ed.v; 
        if(vis[j])continue;
        int s=dfs(j);
        size = max(size,s);
        sum += s;
    }
    ans = min(ans, max(size, n-sum));
    return sum;
}
```



- 邻接表建树

```cpp
#include <iostream>
#include <vector>
using namespace std;
const int N = 1e5+5;
struct edge{int v,w;};
vector<edge> e[N];
int vis[N];
int n;
int main(){
    cin>>n;
    for(int i=1;i<n;i++){
        int u,v,w;cin>>u>>v>>w;
        e[u].push_back({v,w});
        e[v].push_back({u,w});
    }
    dfs(1);
    
    return 0;
}
```



## 倍增法求树的重心

​	预处理一下各节点的重儿子，就可以**用倍增法快速向下跳**，判断那个点是重心。
