## 倍增算法

> 最经典的求LCA算法



- **dep[u]**存u点的深度

- **fa[u] [i]** 存从u点向上跳 2(i) 层的祖先节点（ 2(i)表示2的 i 次幂 ，i = 0,1,2,3,....)

  > 例如，从节点 9 向上 跳 2(0) 层的祖先是 3 ，跳 2(1) 层的祖先是 6，跳 2(2) 层的祖先是1
  >
  > 跳 2(3) 的祖先是 0 (哨兵)

## 算法过程

### 1. 先DFS一遍，创建 ST 表

​	**倍增递推**，fa[u] [i] = fa[ **fa[u] [i-1]** ] [i-1]

### 2.利用 ST 表求LCA



```cpp
const int N = 5e5+10;//500000+10
vector<int> e[N];//建图，存放与每个点直接连接的点
int dep[N],fa[N][20];
//这里开20是因为，2(19)=524288,相当于50万层，已经很大了，当然可以根据题目来设定。

// O(nlogn)
void dfs(int u,int father){
    dep[u]=dep[father]+1;//更新深度
    fa[u][0]=father;
    //创建 ST 表
    for(int i=1;i<=19;i++){// 这里是被看作logn的
    	fa[u][i]=fa[fa[u][i-1]][i-1];
    }
    //枚举下一个点作为子节点
    //注意要抛开 fa 节点
    for(int v:e[u])
        if(v!=father)dfs(v,u);
}
// O(logn)
int lca(int u,int v){
    if(dep[u]<dep[v])swap(u,v);//确保u的深度比v大
    //先让u跳到和v同一层
    for(int i=19;i>=0;i--){
        if(dep[fa[u][i]]>=dep[v])u=fa[u][i];
    }
    if(u==v)return v;
    for(int i=19;i>=0;i--){
        if(fa[u][i]!=fa[v][i]){
            u=fa[u][i];v=fa[v][i];
        }
    }
    return fa[u][0];
}
```
