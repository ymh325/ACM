# 树上启发式合并

> DSU on Tree



## 简介

​	树上启发式合并是一种 **高效处理“静态子树查询”**的算法。所谓“静态子树查询”，指的是我们需要回答关于一棵树中每个**子树**（通常是所有子树）的**某种统计量查询**，且在每个节点上的询问互不干扰。

> 这里的“静态”强调树的结构和查询内容不会改变



## 问题引入：

​	**你面临这样一个情境：**

1. **你有一棵有根树**（通常以1号节点为根）。
2. **每个节点都有一个权值**（可以是颜色、数字、字符串等）。
3. **你需要回答关于大量子树的统计问题**。例如：
   - 子树内不同颜色的节点数。
   - 子树内出现次数最多的权值（即众数）。
   - 子树内所有权值的mex（最小未出现值）。
   - 子树内满足某种条件的节点对数。



## **树上启发式合并的特征：**

1. 没有修改操作
2. 可以通过遍历子树，建立信息统计，得到所有查询的答案



## 树上启发式合并的过程：

`void dfs(u,keep)`，`u` 表示当前节点，`keep` 表示是否保留子树 `u` 对信息的贡献

1. **先遍历**所有**轻儿子的子树**，遍历结束时，消除对信息的贡献，`dfs(轻儿子,0)`

2. **再遍历**唯一**重儿子的子树**，遍历结束时，保留对信息的贡献，`dfs(重儿子,1)`

3. 考察单个节点 `u` ,对信息进行贡献

4. **再遍历所有轻儿子**的子树上面的每个节点，都**重新对信息进行贡献**

5. 得到子树 `u` 的答案

6. 如果 `keep == 0`，**对于父亲节点**来说，**消除**此子树 `u` 的贡献

   如果 `keep == 1`，**对于父亲节点**来说，**保留**此子树 `u`的贡献


## 代码演示

此处维护的信息是 每个子树（含根节点）的所有节点总共有多少个不同的权值

```c++
#include <iostream>
#include <vector>
#define OFF ios::sync_with_stdio(false);cin.tie(0);
using namespace std;
const int N = 1e5+5;
vector<int> e[N];
int fa[N],son[N],sz[N];
int cnt[N],col[N],res[N],mx;//* col[i]指代 节点权值
void get_son(int u,int fat){
    fa[u] = fat,sz[u] = 1;
    for(int v:e[u]){
        if(v == fat)continue;
        get_son(v,u);
        sz[u]+=sz[v];
        if(sz[son[u]]<sz[v])son[u]=v;
    }
}
void add(int u,int son){
    cnt[col[u]]++;
    if(cnt[col[u]]==1)mx++;

    for(int v:e[u]){
        if(v==fa[u]||v==son)continue;
        add(v,son);
    }
}
void cancle(int u){
    cnt[col[u]]--;//* cnt[col[u]] = 0;
     for(int v:e[u]){
        if(v!=fa[u])cancle(v);
    }
}
void dfs(int u,int keep){
    for(int v:e[u]){
        if(v==fa[u]||v==son[u])continue;
        dfs(v,0);
    }
    if(son[u])dfs(son[u],1);
    
    add(u,son[u]);//* 重新加上 u节点 及其 轻子树的贡献
    res[u] = mx;
    if(!keep)cancle(u),mx=0;//* 如果不保存,则清空这颗子树的贡献
}
int main(){
    OFF
        
	/*其他的操作*/

    return 0;
}
```
