## Problem Description


果园之中，累累硕果已被悉数打下，按照品种分门别类，聚作了 nn 堆果子，每堆果子都有着各自的质量。

超哥打算将这 $n$ 堆果子，最终合并为完整的一堆。

**这些果堆首尾相连、排成环形，第一堆与最后一堆同样互为相邻。**

他每次只能选取**相邻的两堆果子**进行合并：

- 合并所消耗的体力，为这两堆果子质量的较大值；
- 合并后新果堆的质量，也取这两堆质量中的较大值。

历经 n−1n−1 次合并，所有果子终将归为一堆。

超哥希望耗费的总体力尽可能小，请你帮他算出：**合并完所有果子所需的最小体力消耗。**

## Input

第一行输入一个整数 $T$ (1≤$T$≤$10^4$) 表示数据组数，对于每组数据：

- 第一行输入一个整数 $n$（1≤$n$≤$10^5$），代表果子堆数。
- 第二行输入 $n$ 个用空格隔开的整数，第 $i$ 个整数代表第 $i$ 堆果子的质量 $a_i$​（1≤$a_i$≤$10^9$）。

**保证所有测试样例的 $n$ 之和不超过 $10^5$。**

## Output

对于每个测试样例，输出一行一个整数，表示将 $n$ 堆果子合并成一堆所消耗的最小体力。

## 输入数据 1

```input1
1
3
4 5 6
```

## 输出数据 1

```output1
11
```


# 思路
将环上每对相邻堆的合并代价看作边权，最终一定会选出 条边形成一棵“生成树”。为了使 总代价最小，应删除权值最大的那条边，因此答案为所有相邻最大值之和减去其中最大的一项。

```cpp
#include <iostream>
#include <vector>
using namespace std;
#define OFF ios::sync_with_stdio(0),cin.tie(0);
#define int long long
void solve(){
    int n;cin>>n;
    int mx=0,res=0;

    vector<int> a(n);

    for(int i=0;i<n;i++){
        cin>>a[i];
    }
    for(int i=0;i<n;i++){
        int j=(i+1)%n;

        mx = max(mx,max(a[i],a[j]));
        res+=max(a[i],a[j]);
    }
    cout<<res-mx<<'\n';
}
signed main(){
    OFF
    int t;cin>>t;
    while(t--){
        solve();
    }  
    return 0;

}
```
