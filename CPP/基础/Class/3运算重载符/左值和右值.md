### 左值和右值

- **左值**

  左值是指可以出现在**赋值语句左边**的表达式，它表示一个具名的、**有明确存储地址**的对象。左值通常有较长的

  生命周期，直到超出其作用域才会被销毁。例如：

```cpp
int a = 10;  // a 是左值，因为它有一个具体的内存地址，可以被赋值和取地址
int* ptr = &a;  // 可以取 a 的地址
```

- **右值**

  右值是指只能出现在**赋值语句右边**的表达式，它通常是临时的、**没有明确存储地址**的对象。右值可以分为纯右值（prvalue）和将亡值（xvalue）。

  - 纯右值：：一般是常量、临时对象或者字面量等。

    ```cpp
    int b = 20 + 30;  // 20 + 30 是纯右值，它是一个临时的计算结果，没有具体的名称
    int c = 5;  // 5 是纯右值，是一个字面量
    ```

  - 将亡值：通常是通过 `std::move()` 转换得到的，它表示一个资源可以被移动的对象，即将要被销毁。

  - ```cpp
    std::vector<int> vec1 = {1, 2, 3};
    std::vector<int> vec2 = std::move(vec1);  // std::move(vec1) 是将亡值
    ```



右值是不能被非 `const` 引用绑定的。这是 C++ 语言的一个规则，目的是防止对临时对象进行意

外的修改，因为临时对象的生命周期通常很短暂，在表达式结束后就会被销毁。



### 为什么 C++ 规则中要防止对临时对象进行意外的修改？

临时对象的生命周期通常很短暂，它们在创建它们的表达式结束后就会被销毁。

如果允许非 `const` 引用绑定到临时对象，并且在绑定后对其进行修改，那么**这些修改可能没有实际意义**，因为临时对象很快就会消失。
