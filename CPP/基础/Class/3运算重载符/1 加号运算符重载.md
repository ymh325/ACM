## **运算符重载**

> 运算符重载概念：对已有的运算符重新进行定义，赋予其另一种功能，以适应不同的数据类型
>



**C++ 规定赋值（`=`）、下标（`[]`）、调用（`()`）和成员访问箭头（`->`）等运算符必须被定义为类成员操作**。将赋值操作符重载为成员函数，是 C++ 编程中的约定和最佳实践，有助于增强代码的可读性和可维护性，也让其他开发者更容易理解代码逻辑 。



**所有的操作运算符，都可以重载**，**包括cout，cin**



### 加号运算符重载

> 两种重载方式：成员函数重载，全局函数重载
>

作用：实现两个自定义数据类型相加的运算

```c++
class per{
public:
    int a;
    int b;
    
    //函数内部相当于自定义了一个类成员加法运算的规则，这个规则可以任意制定
    per operator+(const per& p){//成员函数重载
        per temp;
        temp.a = this->a+p.a;
        temp.b = this->b+p.b;
        return temp;
    }
}
per operator+(int x){//全局函数重载
        per temp;
        temp.a =this->a+x;
        temp.b =this->b+x;
        return temp;
    }
int main(){
    per p1;per p2;
    p1.a=10;p1.b=10;
    p2.a=20;p2.b=20;
    
    
    per p3 = p1+p2 //相当于p3 = p2.operaor+(p1)
    per p4 = p1+10 //相当于p4 = operaor(p1,10)
    return 0;
}
```
