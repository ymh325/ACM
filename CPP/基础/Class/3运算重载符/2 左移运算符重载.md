### 左移运算符重载

> 理论上来说有两种方式，但利用成员函数得不到我们想要的形式，所以只能利用全局函数
>

作用：输出自定义的数据类型

```c++
#include "iostream"
using namespace std;
class per{
public:
    int ma=1;
    //利用成员函数重载 左移运算符  p.operator<<(cout)  简化版本 p << cout
    //不会利用成员函数重载<<运算符，因为无法实现 cout 在左侧
    void operator<<(ostream &cout){}
};
//只能运用全局函数重载左移运算符
void operator<<(ostream &cout,per&p){
    cout<<p.ma;
};

//当需要cout<<p<<endl（实现链式编程）时，需要返回值ostream
//并且要引用&，因为ostream只有cout这一个对象，只能由cout自己来接收自己
ostream& operator<<(ostream &cout,per&p){
    cout<<p.ma;
    return cout;
} 
int main() {
	per p;
    cout<<p;//左移运算重载函数 不需要 返回值ostream&
    cout<<p<<endl;//左移运算重载函数 需要 返回值ostream&
    return 0;
}
```
