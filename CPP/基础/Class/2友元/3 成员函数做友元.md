### 成员函数做友元

让一个类的成员函数可以访问另一个类的私有属性或函数

```cpp
class per;
class gay{
public:
    per* p;
    gay();
    void visit();
    ~gay();
};

class per{
    friend void gay::visit();
private:
    int a;
};

gay::gay(){
    p = new per;
    p->a=1;
}
gay::~gay(){delete p;}

void gay::visit(){
    cout<<p->a;
}

int main(){
    gay p;
    p.visit();
    return 0;
}
```

> #### 注意
>
> 在class gay中使用class per时，需要**确保per的完整定义对gay可见**，也就是gay中与per有关的函数需要写在per下面。
> 将class per的定义提前到class gay之前，或在class gay中包含per.h头文件（如果per定义在单独的头文件中）。
> 确保per的析构函数为public且显式声明为虚函数（若per可能作为基类）。
