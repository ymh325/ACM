### 继承同名成员的处理方式



**问题：** 当子类与父类出现同名的成员，如何通过子类对象，访问到子类或父类中同名的数据呢？



- 访问**子类**同名成员，**直接访问**即可

- 访问**父类**同名成员，需要**加作用域**

  

```c#
class Base{
public:
    int a;
    Base(){a=100;}
};

class Son{
public:
    int a;
    Son(){a=200;}
}

// 同名的成员属性
void text01(){
   Son s;
    cout<<"Son a:"<<s.a<<endl; //访问子类的a
    cout<<"Base a;"<<s.Base::a<<endl;// 访问父类的a
}
// 同名的成员函数
void 

int main(){
    text01();
}
```
