### 继承中构造和析构顺序



子类继承父类后，当**创建子类对象**，也会**调用父类的构造函数**



**问题：** 父类和子类的构造和析构顺序是谁先谁后？



```c#
class Base{
public:
    Base(){cout<<"Base构造函数！"<<endl;}
    ~Base(){cout<<"Base析构函数！"<<endl;}
};
class Son{
public:
    Son(){cout<<"Son构造函数！"<<endl;}
    ~Son(){cout<<"Son析构函数！"<<endl;}
}
void text01(){
    Son a;
    // 输出
    // Base构造函数！
    // Son构造函数！
    // Son析构函数！
    // Base析构函数！
}

int main(){
    text01();
}
```
