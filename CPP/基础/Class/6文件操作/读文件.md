## 文本文件

#### 第一种

```cpp
char buf[1024]={0};
while (ifs>>buf){
    cout<< buf << endl;
}
```



#### 第二种

```cpp
char buf[1024]={0};
while ( ifs.getline(buf,sizeof(buf)) ){
    cout<< buf << endl;
}
```



#### 第三种

```cpp
string buf;
while ( getline(ifs,buf) ){
    cout << buf << endl; 
}
```



#### 第四种

```cpp
char c;
while ( (c = ifs.get()) != EOF ){
    cout<<c;
}
```
