### 成员变量和成员函数分开存储



在C++中，类内的成员变量和成员函数分开存储

只有非静态成员变量才属于类的对象上



```cpp
class per{
public:
    int a; //非静态成员变量  属于类的对象上
    static int b; //静态成员变量  不属于类对象上
    void func1(){} //非静态成员函数  不属于类对象上
    static func2(){} //静态成员函数  不属于类对象上
};
void test01(){
    per p;
    cout<<sizeof(p)<<end;
    //输出为 4，也就是 非静态变量a 的内存大小
}
int main(){
    return 0;
}

```



当类为空时，创建一个空对象，**编译器会为空对象分配一个字节空间**，是为了区分空对象占内存的位置

```cpp
class per{
    
};
int main(){
    per a;
    cout<<sizeof(a)<<endl;
    //输出为 1 
    return 0;
}
```
