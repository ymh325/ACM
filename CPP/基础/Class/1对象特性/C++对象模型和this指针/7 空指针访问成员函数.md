### 空指针访问成员函数



C++中空指针也是可以调用成员函数的，但是也要注意有没有用到this指针



如果用到this指针，需要加以判断保证代码的健壮性

```cpp
class per{
public:
    int age;
    void func(){
        //此函数没有运用this指针，所以尽管this为空，也不影响函数的调用
        cout<<"我是per类"<<endl;
    }
    void show(){
        //此函数运用this指针，为了不影响函数的调用，需要设立一个if判断
        if(this == NULL){
            cout<<"空指针"<<endl;
            return;
        }
        cout<<this->age<<endl;
    }
};
int main(){
    per *p = NULL;
    p->show();
    return 0;
}
```
