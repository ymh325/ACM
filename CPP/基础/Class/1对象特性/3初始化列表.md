### 初始化列表

**作用：**

C++提供了初始化列表语法，用来初始化属性

**语法：**构造函数（）：属性1（值1），属性2（值2）...{}

```cpp
#include <bits/stdc++.h>
using namespace std;
class per{
public:
    int a{},b{},c{};

    per(int x,int y,int z):a(x),b(y),c(z){<-
        //注意有括号
    ->}
    
    per(const per &p){
        a=p.a;b=p.b;c=p.c;
        cout<<"拷贝构造函数"<<endl;
    }
    ~per(){
        cout<<"析构函数"<<endl;
   	}
};

int main(){
    per p1(1,2,3);
    return 0;
}
```
